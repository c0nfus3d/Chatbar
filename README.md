# IRC Bar #
-----------

Dockable Windows IRC Chat Widget


## Commands ##
- `/connect [server] [port :default 6667]` - Connect to a server
- `/join [#channel]` - Join a channel
- `/nick [nickname]` - Change nickname
- `/msg [nickname] Private Message` - Send a private message
